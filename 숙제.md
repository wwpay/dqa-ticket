  - 컬럼명은 항상 유지, 활성 필터는 하단 뱃지(× 포함)로 분리
  - `×` 클릭 이벤트는 `th-filter-clear` 클래스 기반 위임
- **그룹 헤더 색상**: `--*-mid` 변수 (ww-mid/#dbeafe, mvn-mid/#dcfce7, done-mid/#e5e7eb, hold-mid/#d1d5db)
- **섹션별 th 색상**: `.section-X .ticket-table th` 오버라이드 (bg=*-light, color=섹션 accent)
- **보류 & N/A**: 슬래시 `/` → 앰퍼샌드 `&` (i18n 4개 언어 모두)
- **priority 필드**: detail 페이지에서 readonly input (DnD로만 변경)
- **btn-wjira-link**: 완전 제거
- **ticket-id-wrap**: `flex: 1` (신규 등록 메타 3단 균등 유지)
- **GAS 재배포 필요**: v0.3.0 priority 로직 변경(05aaf2b) 이후 미배포 상태
